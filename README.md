# Youtube-MotionLayout

This is "Youtube Player Clone" using MotionLayout and PierfrancescoSoffritti/android-youtube-player.
이 앱은 유튜브 플레이어 클론으로 모션레이아웃과 PierfrancescoSoffritti/android-youtube-player을 사용했습니다.
